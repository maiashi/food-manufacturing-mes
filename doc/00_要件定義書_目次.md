# 食品製造業MESシステム 要件定義書

## 1. 文書の目的
本要件定義書は、食品製造業におけるManufacturing Execution System (MES) の構築にあたり、
適用される法令・規格を遵守しつつ、生産管理・在庫管理・HACCP衛生管理・トレーサビリティ
を統合的に実現するためのシステム要件を定義する。

## 2. システム概要
- システム名: 食品製造業向けMES (Food Manufacturing MES)
- 対象産業: 食品製造業（レトルト食品・チルド総菜・デリカ・おせち料理）
- 製品カテゴリ: レトルト食品(RC-*)、チルド総菜(TC-*)、デリカ(DL-*)、おせち料理(OS-*)
- 適用法令: 食品衛生法、HACCPに基づく衛生管理義務化、食品表示法等

## 3.5 製品カテゴリ概要

本工場は以下の4つの製品カテゴリを扱う。

| カテゴリ | SKUプレフィックス | 保管温度 | 賞味期限 | 主なCCP |
|---------|-----------------|---------|---------|--------|
| レトルト食品 | RC-* | 常温(15〜25℃) | 製造日+18ヶ月 | 殺菌(F値3.0以上)、密閉検査、冷却速度 |
| チルド総菜 | TC-* | 冷蔵(0〜5℃) | 製造日+7日 | 加熱中心温度、冷却速度、冷藏保管、密封 |
| デリカ | DL-* | 冷蔵(0〜4℃) | 製造日+3日 | 原材料COA確認、洗浄塩素濃度、充填温度、アレルギー表示 |
| おせち料理 | OS-* | 常温+冷蔵混在 | 商習慣:12/31まで | 構成品別CCP + 組み立て時アレルチェック、異物混入 |

## 4. 用語定義

| 用語 | 定義 |
|------|------|
| MES | Manufacturing Execution System - 製造実行システム。生産現場の管理・制御を行う |
| HACCP | Hazard Analysis and Critical Control Points - 危害要因分析重要管理点管理手法 |
| CCP | Critical Control Point - 重要管理点。食品の安全性を確保するために必須の工程 |
| OPRP | Operational prerequisite program - 運用前提プログラム |
| FIFO | First In First Out - 先入先出。古いロットから優先して使用する在庫管理手法 |
| LOT (バッチ) | 一貫した品質を持つ生産または入荷品目のグループ |
| トレーサビリティ | 原材料の出所から製品の使用までの追跡可能性 |
| アレルギー表示特定原材料 | 食品表示法で表示が義務付けられた28品目（旧8品目） |
| JFS規格 | Japan Food Safety Standards - 日本食品安全規格 |
| FSC | Food Supply Chain Network - 食品供給連鎖ネットワーク |

## 5. 法令遵守のスタンス
本システムは、以下の基本方針を貫く:
1. 記録改ざんの防止 - 改竄防止機能（電子署名、タイムスタンプ）により信頼性を確保
2. 完全なトレーサビリティ - 原材料から製品まで全工程を追跡可能にする
3. 即時の回収対応 - アレルギー混入等、緊急時の一斉通知・回収対応を支援
4. データ保管義務遵守 - 法令で定められた期間のデータ保持を強制
