# 食品製造業MES - 品質管理要件

## 1. 目的
食品製造における品質検査・品質記録・不適合品管理の要件を定義する。
HACCPのCCP記録と連動し、製品の出荷判断を支援する。

---

## 2. 機能要件

### 2.1 検査規格マスタ

| ID | 要求項目 | 詳細 |
|----|---------|------|
| QC-001 | 検査区分 | 受入検査/工程内検査/最終検査/定期検査 |
| QC-002 | 検査項目登録 | 外観、温度、重量、pH、水分量、食味等 |
| QC-003 | 規格値の設定 | 上限・下限・目標値の定義（数値または基準コード） |
| QC-004 | 判定基準 | OK/条件OK/NGの閾値設定 |

### 2.2 検査実施行

| ID | 要求項目 | 詳細 |
|----|---------|------|
| QP-001 | 検査指示書 | バッチ/ロット単位で検査を実施する指示を生成 |
| QP-002 | 検査結果入力 | 手動入力または計測器からの自動取込 |
| QP-003 | 外観チェック | 写真添付による記録（異常箇所をマーキング可） |
| QP-004 | サンプリング計画 | JIS Z 9015 / ANSI/ASQ Z1.4 に準拠したサンプルサイズ自動計算 |

### 2.3 出荷判断

| ID | 要求項目 | 詳細 |
|----|---------|------|
| QR-001 | 全検査データ連携 | 最終製品の出荷判断は全てのQCデータを参照 |
| QC-002 | 自動判定/手動判定 | CCP/CQIが全てOKなら自動OK、逸脱ありなら手動判定必須 |
| QC-003 | 出荷制限 | 未承認の製品は出荷指示を発行不可 |

### 2.4 不適合品管理（NCR: Non-Conformance Report）

| ID | 要求項目 | 詳細 |
|----|---------|------|
| NC-001 | NCR発生登録 | 検査NG/工程異常/NCP逸脱を即座に記録 |
| NC-002 | リジェクト品フラグ | 対象ロットの在庫移動・出荷をシステムで阻止 |
| NC-003 | 是正措置要求（CAR） | NCRに関連するCARを発行・追跡 |
| NC-004 | 再検査指示 | 再処理後の再検査を指示・記録 |

### 2.5 品質レポート

| ID | 要求項目 | 詳細 |
|----|---------|------|
| QRT-001 | 検査実績レポート | 期間・製品別合格率の推移 |
| QRT-002 | 不適合品レポート | NCR発生件数・理由別集計 |
| QRT-003 | SPC管理図 | 重要品質特性の管理図（X-bar, R等） |

## 3. サンプル検査計画表（例）

```
製品: レトルトカレー
検査区分: 最終検査
サンプリング: ANSI/ASQ Z1.4 Level II, AQL=2.5
ロットサイズ: 500箱 → サンプル数: 50箱

検査項目:
┌───────┬───────────┬────────────┬────────────┐
│ 項目   │ 規格値     │ 測定方法    │ 頻度        │
├───────┼───────────┼────────────┼────────────┤
│ 外観   │ 異常なし   │ 目視       │ 全数       │
│ 重量   │ 210±5g    │ はかり     │ 10%        │
│ パック密閉性 │ 漏れなし │ 加圧テスト │ 5%         │
│ 中心温度 │ 72℃以上  │ 中心温度計 │ CCP記録と連動 │
│ pH     │ 5.5〜6.5  │ pHメーター │ 1バッチ/回 │
└───────┴───────────┴────────────┴────────────┘
```

## 4. カテゴリ別検査項目・基準（製品カテゴリ固有の品質要件）

### 4.1 レトルト食品の検査項目

| 検査区分 | 検査項目 | 規格基準 | 検査方法 | 頻度 |
|---------|---------|---------|---------|------|
| 原材料 | 水分活性(aW) | aW ≤0.95 | アクリヒ酸法 | 1LOT/回 |
| 原材料 | 中心タンパク質 | ≥10% | クイックテスト | 1LOT/回 |
| 工程内 | 殺菌F値計算 | F≥3.0 | オートクレーブ記録解析 | 全バッチ |
| 工程内 | パック密閉性 | 漏れなし | 加圧漏水テスト(2kg/cm²) | バッチ5%サンプリング |
| 完成品 | 経時劣化試験 | 6ヶ月/12ヶ月で微生物検査 | 培養試験 | 新品種/年1回 |
| 完成品 | 内容量 | 表示±5%以内 | はかり | バッチ3% |

**レトルト特有の注意:** 油脂含有率が高い製品（鶏皮多め等）はF値計算に補正係数を適用

### 4.2 チルド総菜の検査項目

| 検査区分 | 検査項目 | 規格基準 | 検査方法 | 頻度 |
|---------|---------|---------|---------|------|
| 原材料 | COA確認 | 食中毒菌陰性 | 書類審査 | 全回 |
| 工程内 | 加熱中心温度 | ≥75℃、1分保持 | 中心温度計 | 全バッチ |
| 工程内 | 冷却速度 | 60℃→10℃まで4時間以内 | 温度プロファイラー | 全バッチ |
| 工程内 | シール強度 | ≥15N/15mm | テンシロン試験機 | バッチ2回/日 |
| 完成品 | リステリア検査 | 陰性（1g中検出不可） | 培養試験 | 週1回 |
| 完成品 | 内容量 | 表示±3%以内 | はかり | バッチ5% |

**チルド特有の注意:** 冷却速度のプロファイリングは初回製品化時と容器変更時のみ実施（以降は月1回）

### 4.3 デリカの検査項目

| 検査区分 | 検査項目 | 規格基準 | 検査方法 | 頻度 |
|---------|---------|---------|---------|------|
| 原材料 | COA確認 | 食中毒菌陰性、農薬残留基準適合 | 書類審査+必要時分析 | 全回 |
| 原材料 | 受入温度 | ≤10℃（生野菜）、≤5℃（肉類） | 温度计 | 全回 |
| 工程内 | 洗浄水残存塩素 | 0.1〜0.3ppm | 塩素試験紙 | 1時間毎 |
| 工程内 | アレルギー混入チェック | 全品目陰性 | テスト試薬 | ライン切り替え時 |
| 完成品 | 総数検査(TPC) | ≤10⁴ cfu/g | 培養試験 | 日1回 |
| 完成品 | リステリア・サルモネラ | 陰性（25g中検出不可） | 培養試験 | 週1回 |
| 完成品 | アレルギー表示確認 | 表示通り | 外観検査 | 全数 |

**デリカ特有の注意:** 
- 加熱杀菌工程がないため、原材料段階での品質管理が最重要
- リステリアは0℃でも増殖するため、冷蔵温度の実測値を継続的に記録
- アレルギー混入防止のため、ライン切り替え時は全清掃後テスト試薬で確認

### 4.4 おせち料理の検査項目

おせちは構成品それぞれのカテゴリ検査に加えて、以下を追加:

| 検査区分 | 検査項目 | 規格基準 | 検査方法 | 頻度 |
|---------|---------|---------|---------|------|
| 組み立て | 构成品組み合わせ間違い | BOM通り | 外観検査+スキャン | 全数 |
| 組み立て | アレルギー表示整合性 | 構成品BOMと表示が一致 | 書類確認 | 全バッチ |
| 包装 | ギフト箱状態 | 傷み・汚れなし | 外観検査 | 全数 |
| 完成品 | 重量チェック | 表示±5%以内 | はかり | バッチ10回/日 |

**おせち特有の注意:** 複数のカテゴリ（レトルト+デリカ+乾燥物）が混在するため、保管温度区分が重要
