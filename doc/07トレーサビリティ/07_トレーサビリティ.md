# 食品製造業MES - トレーサビリティ要件

## 1. 目的
原材料の出所から製品の最終出荷・消費までの全履歴を追跡するトレーサビリティ機能の要件を定義する。
食品リコール発生時の迅速な対応、FSC（Food Supply Chain Network）への対応、消費者からの問い合わせ対応を支援する。

---

## 2. トレーサビリティの基本方針

### 2.1 前進追跡（フォワードトレース）
```
原材料Lot → 加工工程 → 完成品Lot → 出荷先
```
「この原材料を使った製品はどこに出荷したか？」を追跡する。

### 2.2 後退追跡（バックトレース）
```
完成品Lot → 使用原材料Lots → サプライヤー
```
「この製品の原材料は何来源のものか？」を追跡する。

### 2.3 要求事項（食品リコール法に基づく）
- 製品1個(袋)単位での追跡が可能な場合、より迅速・正確な回収が可能
- システム上はロット単位で追跡可能とし、必要に応じて箱/個体レベルの管理も拡張可能

---

## 3. 機能要件

### 3.1 フォワードトレース（原材料 → 製品 → 出荷先）

| ID | 要求項目 | 詳細 |
|----|---------|------|
| FT-001 | 原材料指定追跡 | ロットIDを指定すると、その原材料を使用した全完成品Lotを取得 |
| FT-002 | 製品連動追跡 | 完成品Lotから使用した原材料Lotsの全履歴を取得 |
| FT-003 | 出荷先特定 | 影響を受けた完成品がどの顧客/配送先に shippedされたかを特定 |
| FT-004 | リコール範囲の算出 | 危険な原材料が含まれる製品の出荷数量・在庫数量を計算 |

**フォワード追跡フロー:**
```
1. 問題発生: 原材料LOT-R-0501に異物混入が発見された
2. システム検索: LOT-R-0501を使用していた全バッチを検索
   → BATCH-20260514-001, BATCH-20260513-001, ...
3. 完成品Lotの特定:
   → LOT-FG-20260514-001, LOT-FG-20260513-001, ...
4. 出荷先の特定:
   → 顧客A（300箱）、顧客B（150箱）
5. 在庫状況の確認:
   → 倉庫CにLOT-FG-20260514-001が20箱残っている
```

### 3.2 バックトレース（製品 → 原材料 → サプライヤー）

| ID | 要求項目 | 詳細 |
|----|---------|------|
| BT-001 | 原料履歴の取得 | 完成品Lotを指定すると、使用された原材料・添加物の全リストを取得 |
| BT-002 | サプライヤー情報連携 | 各原材料のサプライヤー名、ロット番号、COA（分析成績書）を取得 |
| BT-003 | バージョン追跡 | BOMバージョンと実際の使用材料の差異を記録・表示 |

### 3.3 リコール管理

| ID | 要求項目 | 詳細 |
|----|---------|------|
| RC-001 | リコール発令書 | アラート、影響範囲、回収数量、対応方法をまとめた文書を生成 |
| RC-002 | 顧客通知リスト | リコール対象製品の出荷先顧客の一覧（連絡先付き）を生成 |
| RC-003 | 回収進捗追跡 | 各顧客からの回収数量を入力・集計 |
| RC-004 | 厚生労働省報告 | 法定の回収届出書雛形へのデータ連携 |

### 3.4 FSC（Food Supply Chain Network）対応

| ID | 要求項目 | 詳細 |
|----|---------|------|
| FSC-001 | 製品識別コード | GS1規格準拠の識別子（GTIN, Lot No, Expiry Date） |
| FSC-002 | データ交換 | FSC標準フォーマットでのデータ出力（XML/JSON） |

### 3.5 トレーサビリティレポート

| ID | 要求項目 | 詳細 |
|----|---------|------|
| TR-001 | 一括追跡レポート | 指定Lotの全履歴を時系列で表示（PDF出力可） |
| TR-002 | 食品リコール訓練記録 | 年次以上のリコール訓練の実施記録と結果レポート |

---

## 4. トレーサビリティデータの必須項目

製造・流通の各段階で以下のデータを保持する:

```
原材料受入時:
  - 原料名, Lot ID, サプライヤー名, 受入日, 製造日, 有効期限, 規格適合証明書

生産時:
  - 使用原材料Lotsのリスト, バッチID, 生産日, CCP監視記録, 検査結果

出荷時:
  - 完成品Lot ID, 出荷先, 出荷日, 数量, 配送伝票番号
```
